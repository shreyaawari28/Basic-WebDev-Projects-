# Email Registration Form

This project is a simple **Email Registration Form** built using **HTML and CSS**.  
It was created as part of my web development learning practice to understand how to style and structure forms using CSS.

---

## Features
- Built using **HTML for structure** and **CSS for styling**
- Includes input fields for full name, email, password, confirm password, and mobile number
- Uses the `required` attribute for basic validation
- Styled using internal CSS for layout, colors, and spacing
- Buttons for **Register** and **Clear** actions

---

## Tools & Technologies
- HTML  
- CSS (internal styling)  
- Visual Studio Code  
- Git & GitHub  

---

## Purpose
This project helped me practice:
- Creating structured web forms
- Applying CSS for form design and responsiveness
- Using form attributes like `required` and `maxlength`
