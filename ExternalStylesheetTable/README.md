# External Stylesheet Table Formatting

This project demonstrates how to format an HTML table using an **external CSS stylesheet**.  
It displays a simple student details table with alternating row colors and styled headers.

---

## Features
- Uses **HTML** for structure and **CSS** for styling  
- Linked external CSS file (`style.css`)  
- Alternating row colors using `nth-child()` selector  
- Clean and readable table layout
