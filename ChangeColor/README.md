# Change Background Color

This project is a simple **HTML + JavaScript** program that allows the user to change the background color of the webpage by selecting one of the given color options using radio buttons.

---

## Features
- Built using **HTML** and **JavaScript**
- Uses **radio buttons** for user input
- Changes the background color dynamically when a color is selected
- Demonstrates use of the `onclick` event and JavaScript function calls

---

## Tools & Technologies
- HTML  
- JavaScript  
- Visual Studio Code  
- Git & GitHub  

---

## Purpose
This project helped me understand:
- How to handle user input with radio buttons  
- How to use JavaScript functions to manipulate webpage properties  
- How to make webpages interactive without reloading
