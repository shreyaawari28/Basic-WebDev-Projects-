# Student Percentage Calculator

This project is a simple **HTML, CSS, and JavaScript** based web application that accepts a student's basic details and marks for three subjects, then calculates the **total marks** and **percentage** instantly.

---

## Features
- Built using **HTML** for structure, **CSS** for design, and **JavaScript** for logic
- Takes input for student details (name, email, contact)
- Accepts marks for three subjects and validates that marks are within 0â€“100
- Calculates total marks and percentage dynamically
- Displays results instantly without reloading the page
- Includes basic styling for a clean, centered layout
