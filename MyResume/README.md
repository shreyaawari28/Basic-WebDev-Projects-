# Resume Page

This project is a simple HTML-based resume page created to practice basic webpage layout using lists, tables, and images.

## Features
- Built entirely using HTML (no CSS or JS)
- Uses lists and tables for structured content
- Includes image and contact information
- Organized section-wise for better readability

## Tools & Technologies
- HTML  
- Visual Studio Code  
- Git & GitHub  

## Purpose
This project helped me practice:
- Structuring a web page with headings, tables, and lists  
- Creating a basic portfolio layout using pure HTML  
- Uploading and hosting a project on GitHub
