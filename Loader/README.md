## ğŸ§ª How It Works
The loader is a circular `<div>` created using equal height and width with `border-radius: 50%`.  
A thick border is applied on all sides, and one side of the border is given a darker color.

A CSS animation is defined using `@keyframes` that rotates the element from `0deg` to `360deg`.  
This animation is applied to the loader with an infinite duration, creating a continuous spinning effect.

All animation logic is handled using CSS only.
