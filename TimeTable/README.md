# Class Time Table

This project is a simple HTML-based class timetable created as part of my web development learning journey.  
The main purpose was to practice HTML table tags, borders, and alignment while understanding how to organize information neatly on a web page.

---

## Features
- Built entirely using HTML (no CSS or JavaScript)  
- Uses table, tr, th, and td tags to structure the timetable  
- Includes border, cell spacing, and alignment attributes  
- Demonstrates how to organize tabular data visually  

---

## Tools & Technologies
- HTML – for structure and layout  
- Visual Studio Code – as the code editor  
- Git & GitHub – for version control and hosting  

---

## Purpose
This project was created as part of my FSJP basic HTML practice to:
- Learn how to create and style tables using only HTML  
- Practice webpage layout and alignment  
- Get hands-on experience uploading and hosting a project on GitHub
