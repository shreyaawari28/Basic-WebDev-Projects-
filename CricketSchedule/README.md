# Cricket Match Schedule

This project displays a simple cricket match schedule using only HTML.  
It demonstrates the use of tables, headings, and hyperlinks to organize and present information in a clean format.

## Features
- Built entirely with HTML (no CSS or JavaScript)
- Uses table tags to format match details
- Includes links for teams or match details
- Practice project from FSJP HTML basics

This project helped me practice table design, alignment, and link creation in HTML.
