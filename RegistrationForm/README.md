# Registration Form

This project is a simple HTML-based registration form created as part of my web development learning journey.  
It is designed for online placement registration and includes various form input controls.

## Features
- Built entirely using HTML (no CSS or JavaScript)
- Includes text boxes, radio buttons, checkboxes, dropdowns, and file upload
- Demonstrates form structure and layout

## Tools & Technologies
- HTML  
- Visual Studio Code  
- Git & GitHub
